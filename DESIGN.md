# 위키 사이트 설계 문서

## 프로젝트 개요
- **프로젝트명**: F-Wiki
- **유형**: 협업 위키 (팀/커뮤니티)
- **인증**: 공개 위키 (인증 불필요)
- **스택**: Next.js 15, React 19, TailwindCSS 4

---

## 핵심 기능
1. ✍️ **마크다운 에디터** - 실시간 프리뷰
2. 🔍 **검색 및 태그** - 전체 검색, 태그 필터링
3. 📚 **버전 관리** - 변경 이력 추적 및 복원
4. 🔄 **버전 비교** - Diff 뷰어

## 페이지 구조

### 주요 페이지

1. **홈페이지** (`/`)
   - 최근 문서
   - 인기 문서
   - 카테고리 목록
   - 검색 바

2. **문서 보기** (`/wiki/[slug]`)
   - 문서 제목
   - 작성/수정 날짜
   - 태그
   - 마크다운 렌더링된 내용
   - 편집 버튼
   - 이력 버튼

3. **문서 편집** (`/wiki/[slug]/edit`)
   - 제목 입력
   - 태그 입력
   - 마크다운 에디터
   - 실시간 프리뷰
   - 저장/취소 버튼

4. **검색 페이지** (`/search`)
   - 검색 입력
   - 필터 옵션
   - 검색 결과 목록

5. **문서 이력** (`/wiki/[slug]/history`)
   - 버전 목록
   - 변경 날짜/시간
   - 변경 요약
   - 비교/복원 버튼

6. **버전 비교** (`/wiki/[slug]/compare?from=v1&to=v2`)
   - Diff 뷰어
   - 추가/삭제/수정 표시

7. **카테고리 페이지** (`/category/[name]`)
   - 카테고리별 문서 목록

8. **태그 페이지** (`/tag/[name]`)
   - 태그별 문서 목록

## 데이터 모델

### Wiki Document
```typescript
interface WikiDocument {
  id: string
  slug: string
  title: string
  content: string // Markdown
  tags: string[]
  category?: string
  createdAt: Date
  updatedAt: Date
  views: number
  version: number
}
```

### Document Version
```typescript
interface DocumentVersion {
  id: string
  documentId: string
  version: number
  title: string
  content: string
  changeDescription?: string
  createdAt: Date
}
```

### Tag
```typescript
interface Tag {
  name: string
  count: number // 이 태그를 사용하는 문서 수
}
```

### Category
```typescript
interface Category {
  name: string
  slug: string
  description?: string
  count: number
}
```

## UI/UX 가이드라인

### 디자인 원칙
1. **단순성**: 깔끔하고 직관적인 인터페이스
2. **가독성**: 문서 읽기에 최적화된 타이포그래피
3. **접근성**: 키보드 네비게이션, 스크린 리더 지원
4. **반응형**: 모바일, 태블릿, 데스크톱 대응

### 색상 팔레트 (제안)
- Primary: 파란색 계열 (#3B82F6)
- Secondary: 회색 계열 (#64748B)
- Success: 초록색 (#10B981)
- Warning: 노란색 (#F59E0B)
- Danger: 빨간색 (#EF4444)

### 타이포그래피
- 제목: Sans-serif (예: Inter, Pretendard)
- 본문: Serif 또는 Sans-serif
- 코드: Monospace (예: Fira Code, JetBrains Mono)
